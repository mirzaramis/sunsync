# SunSync Website

A responsive **React + Vite + TailwindCSS** frontend for **SunSync Hardware**.  
This project provides an engaging website with a multi-step **solar quote request form**, **cost estimation calculator**, and **EmailJS integration** to send requests directly to the sales team.

---

## Features

-  **Modern UI** built with React, Vite, and TailwindCSS  
-  **Multi-step quote form** (Personal Info, Project Details, Preferences, Review & Submit)  
-  **EmailJS integration** – sends quote requests directly to sales@sunsynchardware.com  
-  **Quick cost calculator** – estimates project costs with federal tax credit  
-  **Fully responsive design** – optimized for desktop and mobile  
-  Styled with TailwindCSS for consistent design
